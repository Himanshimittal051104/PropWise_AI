## â–¶ How to run (Frontend)

Clone the repo and move inside folder:
```bash
git clone https://github.com/Himanshimittal051104/RentWise-AI.git
cd RentWise-AI
```

Install packages:
```bash
npm install
```

Start frontend:
```bash
npm run dev
```

Open in browser:
```text
http://localhost:3000
```

---

## â–¶ How to run (Backend)

Go to backend folder:
```bash
cd backend
```

Create virtual environment (optional but recommended):
```bash
python -m venv venv
```


Activate it:
**Windows**
```bash
venv\Scripts\activate
```
**Mac/Linux**
```bash
source venv/bin/activate
```

Install dependencies:
```bash
pip install -r requirements.txt
```

Start backend server:
```bash
python app.py
```

Backend will run at:
```text
http://127.0.0.1:8000
```
